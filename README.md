# SEP Email Processor Sample (Bulk Sender)

This is a simple Python script that acts as a basic email processor for sending bulk emails from a list. It's designed for developers building enterprise email infrastructure.

## Requirements
- Python 3.x
- No additional libraries needed (uses built-in smtplib and csv)

## Usage
1. Create a `recipients.csv` with emails in the first column.
2. Replace SMTP and email details.
3. Run the script: `python script.py`

The script reads recipients and sends customized emails.
